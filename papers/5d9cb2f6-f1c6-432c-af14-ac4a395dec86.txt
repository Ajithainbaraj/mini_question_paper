PART A – MCQs (10 × 1 = 10 Marks)
------------------------------------------------------------
1. Which data structure is primarily used to organize objects in a 3D scene hierarchically, allowing for efficient transformations and rendering?
   A. ) Hash Map
   B. ) Scene Graph
   C. ) Adjacency List
   D. ) Binary Search Tree
2. In 2D game development, which transformation operation changes the size of an object without altering its shape?
   A. ) Translation
   B. ) Rotation
   C. ) Scaling
   D. ) Shearing
3. What is the primary goal of "Game Balancing" in game design?
   A. ) To ensure all characters have the same abilities
   B. ) To make the game equally difficult for all players
   C. ) To create a fair and enjoyable challenge that avoids frustration or triviality
   D. ) To minimize the number of bugs in the game code
4. Which concept is used to model AI behavior by defining a set of states and transitions between them based on specific conditions?
   A. ) Neural Network
   B. ) Decision Tree
   C. ) Finite State Machine
   D. ) Genetic Algorithm
5. In the context of ML in game development, which learning paradigm involves an agent learning optimal actions through trial and error by maximizing a reward signal?
   A. ) Supervised Learning
   B. ) Unsupervised Learning
   C. ) Reinforcement Learning
   D. ) Transfer Learning
6. What is the main advantage of Hardware Rendering over Software Rendering in modern game engines?
   A. ) Lower power consumption
   B. ) Greater control over pixel-level details
   C. ) Significantly faster processing of complex graphics due to dedicated GPUs
   D. ) Easier debugging of rendering errors
7. Which project management technique uses a single time estimate for each activity and focuses on identifying the longest path through the project network?
   A. ) PERT
   B. ) Gantt Chart
   C. ) Critical Path Method (CPM)
   D. ) Scrum
8. A visual tool that displays project schedules as horizontal bars, showing start and end dates of tasks and their dependencies, is known as a:
   A. ) Flowchart
   B. ) PERT Chart
   C. ) Gantt Chart
   D. ) Workflow Diagram
9. Which Pygame module is primarily used for drawing shapes, text, and images onto the screen surface?
   A. ) pygame.mixer
   B. ) pygame.event
   C. ) pygame.display
   D. ) pygame.draw
10. When designing a game avatar in Pygame, what is a crucial aspect to consider for its visual representation and interaction?
   A. ) Server latency
   B. ) Network bandwidth
   C. ) Sprite sheets and animation frames
   D. ) Database schema
